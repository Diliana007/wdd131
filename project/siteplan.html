🩷💜 Board Game Haven 🩷💜

Where Fun Meets Community

1. Site Name

Board Game Haven: A place where people can enjoy tabletop games, good food, and a friendly atmosphere.

2. Site Purpose

This website will showcase the café, provide a remarkable board game library, display upcoming events, offer an online store, allow reservations, and foster community engagement through a forum.

3. Scenarios

  • A parent wants to find family-friendly games and check operation hours before visiting.
  • A gamer wants to book a table for a tournament night and browse available games filtered by difficulty.

4. Planned Content

  • Homepage with intro, opening hours, and a welcoming message.
  • Searchable board game library (filterable by difficulty, duration, and genre).
  • Digital food and drink menu with dietary filters.
  • Online reservation form with date/time selection.
  • Community forum for user reviews and discussions.

5. Color Palette

Colors chosen for a warm, inviting look:

  • Berry Pink: #d16ba5 (Headers, buttons, accents)
  • Soft Purple: #c777b9 (Backgrounds)
  • Light Pink: #fff0f6 (Readable contrast)
  • Dark Text: #3a003a (Readable content)

6. Typography

  • Poppins: For headings and navigation.
  • Roboto: For body text and paragraphs.
